Key Design Decisions

Streaming Audio

Bidirectional WebSocket streams. Deepgram processes audio chunks in real-time. TTS streams back as generated. Result: <1.8s perceived end-to-end latency.

Stateful Sessions

Redis stores dialog state with 30-min TTL. Enables multi-turn context, callback scenarios, and mid-call language switching without losing conversation history.

Graceful Degradation

If AI services fail → IVR fallback menu. If Zenchef offline → collect info, staff follows up. If STT fails → "Sorry, could you repeat that?" with retry. No call ever dropped.

Zenchef API Integration

Full REST API integration for real-time reservation management and guest profile synchronization.

🔍

Check Availability

GET /availability — Query open slots by date, time, party size. Returns available time windows with remaining capacity per slot.

➕

Create Booking

POST /reservations — Creates new reservation with guest details, party size, special requests, dietary notes. Returns confirmation ID.

✏️

Modify Booking

PATCH /reservations/:id — Updates date, time, party size, or special requests. Validates new slot availability before confirming change.

❌

Cancel Booking

DELETE /reservations/:id — Cancels with optional reason. Triggers SMS notification. Frees slot in real-time for other bookings.

API Integration Code

zenchef-client.ts
class ZenchefClient {
  private baseUrl = 'https://api.zenchef.com/v1';
  private restaurantId = 'el-barrio-lux';

  async checkAvailability(date: string, partySize: number) {
    const res = await fetch(
      `${this.baseUrl}/restaurants/${this.restaurantId}/availability`
      + `?date=${date}&covers=${partySize}`,
      { headers: { 'Authorization': `Bearer ${this.apiKey}` } }
    );
    return res.json(); // { slots: [{time, available, capacity}] }
  }

  async createReservation(data: ReservationPayload) {
    const res = await fetch(
      `${this.baseUrl}/restaurants/${this.restaurantId}/reservations`,
      {
        method: 'POST',
        headers: {
          'Authorization': `Bearer ${this.apiKey}`,
          'Content-Type': 'application/json'
        },
        body: JSON.stringify({
          date: data.date,       // "2025-01-18"
          time: data.time,       // "20:00"
          covers: data.partySize,// 4
          guest: {
            firstName: data.name,
            phone: data.phone,
            email: data.email,
            notes: data.specialReq
          }
        })
      }
    );
    return res.json(); // { id, confirmationCode, status }
  }

  async cancelReservation(bookingId: string, reason?: string) {
    return fetch(
      `${this.baseUrl}/reservations/${bookingId}`,
      { method: 'DELETE', body: JSON.stringify({ reason }) }
    );
  }
}
